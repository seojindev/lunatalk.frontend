# Lunatalk.FrontEnd

## lunatalk Shop Front-End Source.

---
## Local Develop Server.
```bash
# cp project/develop.env project/local.env
```

## Local Develop Server.
```bash
# yarn start
# yarn start:dev
# yarn start:prod
```

## Local build.
```bash
# yarn build
# yarn build:dev
```

## build test
```bash
# yarn build:test
```
